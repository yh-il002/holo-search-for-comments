# 技術選定書 (Technical Selection) - 完全無料・ローカル版

YouTubeの字幕データを活用し、APIキーなしで動作する完全無料・ローカル完結型のRAGシステムの技術選定です。

## 1. 方針
- **完全無料・ローカル実行**: OpenAI等の外部有料APIを一切使用せず、全ての処理をローカルマシンのCPU/GPUで完結させます。
- **日本語対応**: 日本語の字幕データを高精度にベクトル化できるモデルを採用します。

## 2. 選定技術一覧

### 2.1 YouTube字幕取得
- **`youtube-transcript-api`**:
    - **理由**: YouTube公式APIキーが不要。動画IDから直接タイムスタンプ付きの字幕リストを無料で取得可能。

### 2.2 テキスト処理・RAGフレームワーク
- **`langchain-huggingface`**:
    - **理由**: HuggingFaceのオープンソースモデルをLangChainから手軽に呼び出すために使用。

### 2.3 埋め込みモデル (Embeddings)
- **`intfloat/multilingual-e5-small`**:
    - **理由**: 
        - **高性能**: 多言語（日本語を含む）のベクトル検索において、非常に高いベンチマークスコアを持つ。
        - **軽量**: モデルサイズが小さく、GPUがない一般的なノートパソコンのCPUでも高速に動作。
        - **無料**: オープンソースであり、API使用料が発生しない。

### 2.4 ベクトルデータベース (Vector DB)
- **ChromaDB (`chromadb`)**:
    - **理由**: Pythonライブラリとして完結。追加のサーバー構築なしでローカルディレクトリにデータを永続化可能。

## 3. 実行環境
- **Python 3.10+**
- **インターネット接続**: 初回のモデルダウンロード時、およびYouTube字幕取得時に必要。処理自体はオフラインでも可能。
